Flexbox & other

Use next classes for excellent dev experience:
  • flex - block to Flexbox
  • inline - block to inline-flexbox
  • Window - Window object of RoxUI
  • Window maxsize - use 2 classes for Max Size Screen
  • center - center block on Flexbox
  • center:horizon - center horizontal block on Flexbox
  • dir:column - flex direction to column
  • padding:5 - use standart padding for RoxUI
  • marginHorizon:5 - use standart margin for RoxUI
  • rox-compose - margin block for UX RoxUI
  • rox-column - cut screen to 2 max-screen blocks
  • rox-header - use standart Header RoxUI
  • h1 *selector
  • h3 *selector
  • Block *selector
  • ul *selector
  • Quote *selector
  • code *selector
  • section *selector
